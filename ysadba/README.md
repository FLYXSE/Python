# Вердомическая усадьба — Telegram Квест-Бот

Интерактивный Telegram-бот с квестом по исторической Вердомической усадьбе графа Толочко. Бот включает 6 локаций с фотографиями, голосовыми подсказками и вопросами на знание истории усадьбы.

## Структура квеста

| Локация | Название |
|---------|----------|
| 1 | Дворец |
| 2 | Липовая аллея |
| 3 | Усыпальница (часовня) |
| 4 | Здание спиртзавода |
| 5 | Амбар и хозяйственные постройки |
| 6 | Парк и водная система |

## Требования

- Python 3.10+
- Telegram Bot Token (от [@BotFather](https://t.me/BotFather))

## Установка

1. Клонируйте репозиторий:
```bash
git clone <repo-url>
cd ysadba
```

2. Создайте виртуальное окружение и установите зависимости:
```bash
python -m venv venv
venv\Scripts\activate  # Windows
pip install -r requirements.txt
```

3. Создайте файл `.env` и добавьте токен бота:
```env
BOT_TOKEN=your_telegram_bot_token_here
```

4. Поместите медиафайлы в корень проекта:
   - `start.jpg`, `start.m4a` — приветствие
   - `loca_1.jpg` — `loca_6.jpg` — фото локаций
   - `loca_1.m4a` — `loca_6.m4a` — голосовые описания локаций

## Запуск

```bash
python bot.py
```
